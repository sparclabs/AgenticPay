# Multi-Products + Multi-Seller Environments

This category includes negotiation environments with:
- Single buyer agent
- Multiple products (each seller has their own unique product)
- Multiple seller agents

## Tasks

### Task1: Parallel Two-Seller Per One Product Negotiation

The fundamental parallel multi-seller negotiation environment that manages simultaneous negotiation processes between one buyer and two sellers, where each seller has their own unique product. The buyer negotiates with both sellers in parallel and can choose to make a deal with the seller offering the lower price. This task tests agents' ability to handle competitive negotiation scenarios where multiple sellers compete for the same buyer, with each seller offering a different product that matches the buyer's requirement.

### Task2: Parallel Three-Seller Per One Product Negotiation

This environment manages parallel negotiation between one buyer and three sellers, where each seller has their own unique product. The buyer negotiates with all three sellers simultaneously and can choose to make a deal with the seller offering the lower price. This task challenges agents to handle more complex competitive scenarios with a larger number of competing sellers, each offering different products that all match the buyer's requirement.

### Task3: Sequential Two-Seller Per One Product Negotiation

This environment manages sequential negotiation where the buyer chooses one seller per round to negotiate with. Each seller has their own unique product. The buyer can switch between two sellers and make a deal with either seller. This task tests agents' ability to make strategic decisions about which seller to negotiate with at each round, requiring sequential decision-making and seller selection strategies while comparing different products.

### Task4: Sequential Three-Seller Per One Product Negotiation

This environment manages sequential negotiation where the buyer chooses one seller per round to negotiate with. Each seller has their own unique product. The buyer can switch between three sellers and make a deal with any of the three sellers. This task challenges agents to handle more complex sequential decision-making scenarios with a larger number of sellers to choose from, each offering different products that all match the buyer's requirement.
