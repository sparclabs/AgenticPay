# Only Multi-Buyer Environments

This category includes negotiation environments with:
- Multiple buyer agents
- Single product
- Single seller agent

## Tasks

### Task1: Parallel Two-Buyer Negotiation

The fundamental parallel multi-buyer negotiation environment that manages simultaneous negotiation processes between two buyers and one seller for the same product. The seller negotiates with both buyers in parallel and can choose to make a deal with the buyer offering the higher price. This task tests agents' ability to handle competitive negotiation scenarios where multiple buyers compete for the same product.

### Task2: Parallel Three-Buyer Negotiation

This environment manages parallel negotiation between three buyers and one seller for the same product. The seller negotiates with all three buyers simultaneously and can choose to make a deal with the buyer offering the higher price. This task challenges agents to handle more complex competitive scenarios with a larger number of competing buyers.

### Task3: Sequential Two-Buyer Negotiation

This environment manages sequential negotiation where the seller chooses one buyer per round to negotiate with. The seller can switch between two buyers and make a deal with either buyer. This task tests agents' ability to make strategic decisions about which buyer to negotiate with at each round, requiring sequential decision-making and buyer selection strategies.

### Task4: Sequential Three-Buyer Negotiation

This environment manages sequential negotiation where the seller chooses one buyer per round to negotiate with. The seller can switch between three buyers and make a deal with any of the three buyers. This task challenges agents to handle more complex sequential decision-making scenarios with a larger number of buyers to choose from.
